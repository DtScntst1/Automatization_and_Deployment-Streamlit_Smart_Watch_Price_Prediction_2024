# ⌚ Smart Watch Price Prediction

An end-to-end ML web application predicting smartwatch prices based on battery life, brand, and screen type.

## 💡 Overview
- **Objective:** Construct a predictive pricing model for wearables.
- **Business Value:** Instant market value estimations for consumers and retailers.

## 🛠️ Tech Stack
- **Deployment:** Streamlit
- **Data Science:** Pandas, Scikit-Learn
