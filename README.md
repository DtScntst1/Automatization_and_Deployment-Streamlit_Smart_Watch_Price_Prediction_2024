# Smart Watch Price Prediction

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python&logoColor=white)
![Streamlit](https://img.shields.io/badge/Streamlit-App-FF4B4B?logo=streamlit&logoColor=white)

![App Mockup](https://files.catbox.moe/v02yt2.png)

An end-to-end Machine Learning web application deployed via Streamlit, designed to predict the price of smartwatches based on technical features like battery life, brand, and screen type.

## Overview
Objective: Analyze e-commerce data and construct a predictive pricing model for wearables.
Business Value: Provides instant market value estimations for both consumers and retailers.

## Tech Stack
Deployment: Streamlit
Data Science: Pandas, Scikit-Learn
